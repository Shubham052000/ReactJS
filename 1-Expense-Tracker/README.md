## Run Locally

Clone the project

```bash
  git clone https://github.com/Shubham052000/ReactJS.git
```

Go to the project directory

```bash
  cd 1-Expense-Tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
